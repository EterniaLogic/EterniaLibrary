{\bfseries{Artificial Intelligence}} \mbox{[}/\+AI/\mbox{]}\+:
\begin{DoxyItemize}
\item FSM AI -\/ Finite State machine AIs follow basic brain mechanics.
\item Advanced Thinker -\/ Comparable to FSM AIs, yet also allow for multiple inputs and outputs, the \textquotesingle{}epoch\textquotesingle{} of self-\/learning AIs whether it be Sin functions all the way to image and video processing. Not necessarily more powerful than FSM AIs, only useful for specific applications.
\item Self-\/learning controller (Research specific) -\/ sets goals and modifies the AI to meet the demands, over time will become what the user demands from a single entity.
\item Genetic controller (Application specific) -\/ sets a user-\/specific goal and \textquotesingle{}kills\textquotesingle{} off a set number of simulated AIs until the goal is met, randomly creating a set and cross-\/breeds the best solutions of AI cores.
\end{DoxyItemize}

{\bfseries{Application Program Interface}} \mbox{[}/\+API/\mbox{]}\+:


\begin{DoxyItemize}
\item Core API system to allow multiple languages to cross-\/talk with shared memory
\item Permissions for different users and API modules

Not to be confused with a Web API, which interfaces a server\textquotesingle{}s language with a client of a different language. This API is to enable multiple languages to be connected with the core program.
\end{DoxyItemize}
